Desktop Linux

With usage of Nix package manager: 
          nix-env -iA ghc.ergvein-wallet -f https://github.com/hexresearch/ergvein/archive/master.tar.gz
We use binary cache provided by Cachix to speedup installation. Run the following commands to enable it: 
          
          nix-env -iA cachix -f https://cachix.org/api/v1/install
          cachix use ergvein
